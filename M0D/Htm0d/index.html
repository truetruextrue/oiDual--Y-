As Três Classes de Código


      

        
⌂

        

          
K · Python / Bash / SQL / Lógica → TEMPO

          
Processamento sequencial, ação no tempo, lógica pura. Python age linha por linha — ele existe no fluxo temporal. Mapeia para: V = freq(len) S = binário χ = funções−imports+returns

        

      


      

        
≈

        

          
B · HTML / CSS / JSON → ESPAÇO

          
Estrutura visível, interface, o que o olho toca. HTML não processa — ele contém e organiza no espaço. Mapeia para: V = tags→freq E = estilos χ = tags−scripts+css

        

      


      

        
△

        

          
K+B juntos → SISTEMA CONVERGENTE ou DIVERGENTE

          
Quando dois scripts se combinam, o Δ mede interferência. Dois processadores de dados = saturação. Python→HTML = convergência ideal. Dois scripts de UI = divergência. Ciclo fechado (K processa output de B) = Toro T² · χ=0.
